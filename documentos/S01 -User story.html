 Descrição 

 

Funcionalidade que permite que o usuário cadastrado realize a compra com apenas um clique, utilizando um método de pagamento e endereço previamente salvos. Isso agiliza o processo de checkout sem necessidade de preencher as informações a cada compra. 

 

User Stories 

Eu, como cliente de E-commerce, 

Quero realizar a compra com um único clique 

Para que eu possa finalizar a compra rapidamente sem passar por todas as etapas do checkout. 

 

Critérios de Aceitação 

  • Através de um botão de compra rápida o usuário deve conseguir realizar a compra com um único clique 

  • Através de um método de pagamento e com um endereço de entrega previamente salvos o usuário deve conseguir realizar a compra rápida 

  • O botão compra rápida deve estar disponível na página do produto. 

  • A configuração rápida de compra deve aparecer antes de finalizar a compra. 

  • Após a compra o cliente deve receber uma mensagem de confirmação com o número de identificação do pedido. 

  • A compra deve ser processada automaticamente com base nos dados do cliente logado. 

  • A compra deve ser salva no banco de dados e gerar a atualização do estoque. 

    Requisitos técnicos 

Backend: 

  • banco de dados 

  • Criar tabela para armazenar métodos de pagamentos 

  • Criar tabela para salvar dados de entrega 

  • Atualizar tabela de pedidos para suportar compra rápida 

  • API 

  • Endpoint para processar compra rápida (POST/comprarapida) 

  • Validação do método de pagamento e endereço antes de confirmar a compra 

  • Atualizar quantidade em estoque automaticamente 

Frontend 

  • Adicionar botão compra rápida na tela de produto 

  • Exibir tela de confirmação do pedido 

  • Exibir modal para confirmação/recusa de compra antes do processamento de pedido 

Possíveis desafios 

Implementação de segurança na API para evitar compras realizadas sem autenticação. 

Tratar casos de concorrência na hora da compra. 

Tratar casos de esgotamento de estoque e falhas na compra. 

Garantir que funcionalidade não permita compras acidentais.
